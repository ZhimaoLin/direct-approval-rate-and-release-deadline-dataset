DESCRIPTION

Implement generic retry helper

Implemented a generic retry helper than can be used to call a
function repeatedly with backoff interval until the function returns
non-error.

Example:

    fn := func(r int) error {
        fmt.Printf("Attempting retry %d\n", r)
        if r == 8 {
            return nil
        }
        return errors.New("failed")
    }

    rh := common.NewRetryHelper(10, time.Second*2, 2, fn)
    err := rh.Run()
