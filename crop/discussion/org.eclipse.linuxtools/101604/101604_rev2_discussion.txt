  to readonlyRootFs and tmpfs for /run and /tmp

  basic security option is chosen by user
- add CapAdd and CapDrop properties support to
  ContainerInspectContentProvider

- fix the reddeer ImageRunSelectionPage to include the
  new modified basic security option text
- modify the BasicSecurityTest to check for the capDrop
  property being set
