[RFC] Add base classes for XY charts based on SWT charts

- Add Base XY Chart Viewer
- Add Bar Chart Viewer
- Add default mouse drag and mouse wheel provider
- Add default selection provider
- Add default tool tip provider
- Add tool tip provider for bar charts
- Add Base chart view
- Support for current time and time range synchronization
- Support for open, select and close trace synchronization
- Add example plug-in with implementation of a histogram

Implementation based on proposal for new histogram with SWT chart
by Braxton Mccarthy (https://git.eclipse.org/r/#/c/9231).
Thanks also to Ivy Mitchell for suggestions on how to avoid
loss of precision when converting long <-> double values.
