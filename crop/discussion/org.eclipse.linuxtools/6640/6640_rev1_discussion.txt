DESCRIPTION

Add support for default launch shortcut provider types.

The launch provider extension point schema has a default attribute, it
provides a way to specify a default plug-in for a profiling type. In the
ProfileLaunchShortcut class, getProfilingProvider finds a default provider;
if none is found it returns the first provider found.
