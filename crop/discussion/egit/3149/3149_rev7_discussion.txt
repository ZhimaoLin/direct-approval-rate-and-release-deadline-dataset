DESCRIPTION

Add diff viewer that uses theme and editors settings.

The DiffViewer class uses theme settings for added,
removed, and hunk lines, and editor settings for
font and foreground, background, selection, and line
highlight colors.  It also supports displaying a line
number column in the overview ruler.
