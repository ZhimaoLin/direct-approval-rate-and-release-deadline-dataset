DESCRIPTION

Target platform definition for SWTBot-based unit tests

'SWTBot for Eclipse Testing' feature should be installed in order to run
SWTBot-based unit tests. SWTBot's download page provides a set of update
sites for different Eclipse releases. We use an update site for
Ganymede [2] as EGit targeted for Eclipse 3.4 (Ganymede) [3].

[1] http://www.eclipse.org/swtbot/downloads.php
[2] http://download.eclipse.org/technology/swtbot/ganymede/dev-build/update-site
[3] http://wiki.eclipse.org/EGit/FAQ#What_is_EGit.27s_target.3F
