DESCRIPTION

[RFC] Change staging view "Compressed" mode to be hierarchical

This is an implementation of the idea here:

https://bugs.eclipse.org/bugs/show_bug.cgi?id=407607#c9

It changes the "compressed" mode, e.g. from this:

* a/b/c/d/x
  * file1.txt
* a/b/c/d/y
  * file2.txt

to this:

* a/b/c/d
  * x
    * file1.txt
  * y
    * file2.txt
