DESCRIPTION

Allow to configure a default target folder for cloning Git Repositories

Cloning Repositories into the workspace location has proven to cause
different issues. This adds a "default folder" Preference that can
be configured and defaults to the user home as obtained from FS.
This folder is then suggested as parent folder in the GitCloneWizard.
